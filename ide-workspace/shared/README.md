# @nu-art/ide-workspace-shared

Serializable layout types for the IDE workspace shell: dock edges, tool panel modes, editor split tree.

See [ide-workspace README](../README.md) for the full spec.

## Exports

- `DockEdge`, `PanelMode`, `ToolPanel`, `EditorTab`, `PaneNode`, `WorkspaceState`
- `IDE_WORKSPACE_SCHEMA_VERSION`
