# @nu-art/archiving-shared

Shared API definitions and types for the archiving feature. Used by archiving-backend and archiving-frontend.

## Deps

- `@nu-art/api-types` — ApiDefResolver, BodyApi, QueryApi, HttpMethod
- `@nu-art/ts-common` — DB_Object, UniqueId

## Exports

- `RequestBody_HardDeleteUnique`, `RequestQuery_DeleteAll`, `RequestQuery_GetHistory`, `ApiStruct_Archiving`, `ApiDef_Archiving`

No `@nu-art/thunderstorm-*` dependency.
