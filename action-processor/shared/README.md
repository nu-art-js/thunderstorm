# @nu-art/action-processor-shared

Shared API definitions and types for the action-processor feature. Used by action-processor-backend and action-processor-frontend.

## Deps

- `@nu-art/api-types` — ApiDef, BodyApi, QueryApi, HttpMethod
- `@nu-art/ts-common` — Minute

## Exports

- `Request_ActionToProcess`, `ActionMetaData`, `ApiStruct_ActionProcessing`, `ApiDef_ActionProcessing`

No `@nu-art/thunderstorm-*` dependency.
