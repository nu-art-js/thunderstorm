# @nu-art/sync-env-shared

Shared API definitions and types for the sync-env feature. Used by sync-env-backend and sync-env-frontend.

## Deps

- `@nu-art/api-types` — ApiDefResolver, BodyApi, QueryApi, HttpMethod
- `@nu-art/ts-common` — Minute, UniqueId

## Exports

- `ApiDef_SyncEnv`, `ApiStruct_SyncEnv`, request/response types, `SyncEnvBackupMetadata`

No `@nu-art/thunderstorm-*` dependency.
