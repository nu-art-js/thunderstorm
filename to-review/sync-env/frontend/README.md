# @nu-art/sync-env-frontend

Frontend API client for sync-env (sync environment backup/restore). Uses HttpClient and sync-env-shared ApiDef; no `@nu-art/thunderstorm-*` dependency.

## Deps

- `@nu-art/sync-env-shared` — ApiDef_SyncEnv, types
- `@nu-art/http-client` — HttpClient
- `@nu-art/ts-common` — Module

## Exports

- `ModuleFE_SyncEnvV2` (API caller vv1)
